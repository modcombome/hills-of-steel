# SEO Audit - Hack Hills Of Steel

## 1. Tổng quan

- URL gốc: https://modcombo.me/hills-of-steel/
- Title gốc: Tải Hack Hills Of Steel (Full Tiền, Kim Cương) V8.7.2
- Title đề xuất: Tải Hack Hills Of Steel v8.7.2 Full Tiền, Kim Cương
- Meta đề xuất: Tải Hack Hills Of Steel v8.7.2 full tiền, vô hạn kim cương. Xem tính năng MOD, gameplay xe tăng, lưu ý cài đặt và cách tải an toàn.

## 2. Điểm SEO hiện tại

Ước tính: 7.3/10

Điểm mạnh:
- Có title, meta description, canonical, OG/Twitter meta.
- Có Rank Math schema.
- Bài có H1, H2, H3 tương đối đầy đủ.
- Có bảng thông tin app, ảnh minh họa và FAQ.
- Nội dung đã bao phủ gameplay, tính năng MOD, mẹo chơi, lưu ý và CTA.

Vấn đề:
- Code HTML quá nặng vì xuất từ WordPress/Flatsome, chứa nhiều CSS/JS/plugin không cần thiết cho GitHub Pages.
- Có nhiều inline CSS dài, jQuery, wpDiscuz, Dashicons, Emoji CSS, admin bar CSS.
- Title gốc ổn nhưng có thể viết tự nhiên hơn và đặt version sát keyword hơn.
- Meta gốc có cụm “đỉnh cao”, “để mở khóa và nâng cấp” hơi quảng cáo; nên chuyển sang ngữ nghĩa rõ hơn.
- FAQ trong schema hơi dài, nên rút gọn để Google dễ hiểu.
- Một số câu khẳng định về bản hack nên viết thận trọng hơn, tránh cam kết an toàn tuyệt đối.
- Bài có nhiều phần liên quan/tác giả/bình luận ở cuối nếu đưa thẳng lên GitHub sẽ làm nhiễu nội dung chính.

## 3. Bản tối ưu đã làm

- Tách code thành static GitHub Pages.
- Giữ nội dung chính nhưng viết lại gọn hơn, tự nhiên hơn.
- Thêm cảnh báo an toàn cho bản MOD/hack.
- Làm lại schema sạch và dễ đọc.
- Giữ ảnh chính + ảnh nội dung quan trọng.
- Thêm mục lục sticky.
- Tối ưu tốc độ bằng CSS/JS nhẹ.

Bản mới phù hợp để đăng lên GitHub Pages, làm landing static hoặc trang entity hỗ trợ SEO cho bài chính. Nếu dùng để hỗ trợ site chính, nên giữ canonical về MODCOMBO. Nếu muốn GitHub Pages tự index, hãy đổi canonical sang URL GitHub Pages.
